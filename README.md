# market cap for bitcoin vs other altcoins

market cap comparison between bitcoin and other altcoins.  

this is a simple program that pulls data from coinmarketcap.com daily and show the data.  this allows us to see the trend of bitcoin marketcap vs all the other altcoins.  


remember to replace mongolab api key in the code with the real mongolab api key.  create the mongo dbname on mongolab management console as well.



deploy:

in the mktcap folder:

[appengine]/appcfg.py -A coinmktcap update ./
